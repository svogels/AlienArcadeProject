ü§î What is a Game Loop?

Imagine you're telling a story that needs to keep going. You might say, "The adventure continues as long as the hero is still searching for the treasure." A while loop in programming is very similar. It tells the computer to keep doing a set of actions "while" a certain condition is true. For our game, we want it to run forever, so we'll say while True!

üñºÔ∏è Visual Metaphor:

Think of a hamster running on its wheel. The wheel goes around and around without stopping. Our game loop is like that wheel. It constantly repeats a cycle of checking for player input, updating the game's state, and drawing everything on the screen. This happens so fast it looks like smooth motion!

üéÆ How We'll Use It in Our Game:

The game loop is the heartbeat of our program. Inside the loop, we will eventually check for key presses, move the player, move the aliens, and check for collisions. For today, its only job is to keep the window open and check if the player has tried to close it.

‚úçÔ∏è Let's Build Our Code, Step-by-Step

Step 1: Import Pygame

First, we need to tell our program that we want to use all the cool features from the Pygame library. The import keyword does exactly that.

import pygame

Step 2: Initialise Pygame

Next, we need to start up all the different parts of Pygame. Calling pygame.init() is like turning the key in a car's ignition. It gets all the Pygame modules ready for us to use.

pygame.init()

Step 3: Create the Game Window

Now we'll create the window where our game will live. We use pygame.display.set_mode() and give it the width and height of our window (800 pixels wide by 600 pixels high). We also set a caption to appear in the title bar.

screen = pygame.display.set_mode((800, 600))
pygame.display.set_caption("Alien Arcade")

Step 4: The Game Loop

This is the most important part. We create a while True: loop that will run forever. Inside this loop, we will check for any events happening, like closing the window.

running = True
while running:
    # This is where all our game logic will go!
    pass # 'pass' is a placeholder that does nothing.

Step 5: Handling the Quit Event

Inside our loop, we need to check if the user has done anything. Pygame puts all user actions into an "event queue". We use a for loop to check every event. If the event is pygame.QUIT (which happens when you click the 'X' on the window), we set our running variable to False to exit the loop.

for event in pygame.event.get():
    if event.type == pygame.QUIT:
        running = False

‚úÖ Full Code for This Activity:

Here is the complete code for today. Copy and paste this into your game_activity_01.py file.

import pygame

# 1. Initialise Pygame
pygame.init()

# 2. Create the game window
screen_width = 800
screen_height = 600
screen = pygame.display.set_mode((screen_width, screen_height))
pygame.display.set_caption("Alien Arcade - Activity 1")

# 3. The Game Loop
running = True
while running:
    # 4. Event Handling
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False

# 5. Quit Pygame
pygame.quit()
