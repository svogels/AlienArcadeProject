  • Challenge 1: Speed Demon

    Find the player_speed variable. Change its value to make the player move much faster or much slower. What happens if you make it a negative number?

  • Challenge 2: Colour Swap

    Change the RGB values for PLAYER_COLOR and ALIEN_COLOR. You can use the color mixer from Lesson 2 to find new colours. Can you make the player a bright green and the alien a hot pink?

  • Challenge 3: Bouncing Alien

    Right now, the player can move off the screen. Look at the code that makes the alien "bounce" off the walls. Can you copy and adapt that logic to stop the player from going past the left and right edges of the window?

  • Challenge 4: A Second Alien

    This one is tricky! Can you create a *second* alien? You would need to create new variables for it (e.g., alien2_x, alien2_y, alien2_speed), update its position in the loop, and draw it to the screen. Can you make it move at a different speed or in a different direction?

  • Challenge 5: Vertical Movement

    Make the alien move up and down instead of side-to-side. You'll need to modify the alien's movement logic to update its `y` position instead of its `x` position, and check if it hits the top or bottom of the screen.

import pygame

pygame.init()

screen_width = 800
screen_height = 600
screen = pygame.display.set_mode((screen_width, screen_height))
pygame.display.set_caption("Alien Arcade - Activity 6")

# Colours
BLACK = (0, 0, 0)
PLAYER_COLOR = (66, 135, 245)
ALIEN_COLOR = (239, 68, 68)

# Player properties
player_width = 50
player_height = 50
player_x = (screen_width - player_width) / 2
player_y = screen_height - player_height - 10
player_speed = 5

# Alien properties
alien_width = 40
alien_height = 40
alien_x = 100
alien_y = 50
alien_speed = 3

running = True
while running:
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False
        if event.type == pygame.KEYDOWN:
            if event.key == pygame.K_LEFT:
                player_x -= player_speed
            elif event.key == pygame.K_RIGHT:
                player_x += player_speed

    # Alien Movement Logic
    alien_x += alien_speed
    if alien_x <= 0 or alien_x >= screen_width - alien_width:
        alien_speed *= -1 # Reverse direction

    # Drawing
    screen.fill(BLACK)
    
    # Draw Player
    player_rect = pygame.Rect(player_x, player_y, player_width, player_height)
    pygame.draw.rect(screen, PLAYER_COLOR, player_rect)

    # Draw Alien
    alien_rect = pygame.Rect(alien_x, alien_y, alien_width, alien_height)
    pygame.draw.rect(screen, ALIEN_COLOR, alien_rect)

    pygame.display.flip()

pygame.quit()
